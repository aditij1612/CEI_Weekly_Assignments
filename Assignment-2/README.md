
# E-Commerce Sales Database Analysis

This project was completed as part of the **Celebal Summer Internship 2026 - Week 2 SQL Task** using **PostgreSQL**.

## Overview
The project analyzes an e-commerce sales database containing customer, product, order, and order item information. SQL queries were used to extract business insights and demonstrate key database concepts.

## Topics Covered
- SQL Basics (SELECT, Constraints, Primary Keys)
- Filtering & Optimization (WHERE, Indexes)
- Aggregation (COUNT, SUM, AVG, GROUP BY, HAVING)
- Joins & Relationships
- CASE Statements
- ACID Properties
- Transactions (COMMIT & ROLLBACK)

## Database Tables
- Customers
- Products
- Orders
- Order_Items
